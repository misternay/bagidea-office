# bagidea

Short alias for **[`bagidea-office`](https://www.npmjs.com/package/bagidea-office)**.

```sh
npx bagidea
```

Installs **BagIdea Office** — your desktop wallpaper goes to work: a living HD-2D office
where every AI agent is a real [Claude Code](https://claude.com/claude-code) session.
Runs the official installer for your platform (downloads a prebuilt app — no Visual
Studio Build Tools / Rust needed on the normal path).
