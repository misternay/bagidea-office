ลากพื้นที่ว่าง = เลื่อน · ล้อเมาส์ = ซูม · ลากจากจุดล่าง (●) ของ node ไปจุดบนของอีก node = เชื่อม · แตกหลายเส้น = ทำขนาน · หลายเส้นเข้า node เดียว = รอให้ครบก่อน · คลิกเส้นเพื่อลบ

🤖 แผนจาก Director

📄 นี่คือ ตัวอย่าง (อ่านอย่างเดียว) — กด 💾 บันทึก เพื่อทำสำเนาที่แก้ไขได้
กด “วิเคราะห์” แล้ว Director จะอ่าน workflow (รวมการแตกแขนง/รวม) แล้วบอกว่าต้องใช้ skill/tool/permission อะไร และมอบหมายให้ใคร
— ยังไม่มีการวิเคราะห์ —
“รันเลย” = ให้ทีมทำตาม workflow นี้เดี๋ยวนี้ แล้วรายงานผล · “สร้างเป็น Skill” = บันทึกเป็นสกิลที่นำไปติ๊กให้ agent ในหน้าตั้งค่า เพื่อเรียกใช้ซ้ำเมื่อไหร่ก็ได้ (หรือพิมพ์สั่ง agent ให้ “รัน <ชื่อ workflow>”)
