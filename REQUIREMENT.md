# REQUIREMENT.md — feature ideas เก็บไว้คัด

ไอเดียฟีเจอร์ที่ "ทำได้จริง + เจ๋ง + ทำให้โปรแกรมโหดขึ้นมาก" — เจ้าของไว้มาคัดเลือก
แต่ละข้อมีเหตุผลและระดับความยาก (🟢 ง่าย · 🟡 กลาง · 🔴 ใหญ่)

> สถานะ: ❌ ยังไม่ทำ · 🚧 กำลังทำ · ✅ ทำแล้ว (อัปเดตเมื่อหยิบมาทำ)

---

## 🤖 ความฉลาดของ Agents

1. **Workflow / Recipe Builder** 🟡 — ต่อ block เป็น flow ("ทุกเช้า → ดึงข่าว → สรุป →
   ส่ง Telegram") แบบ drag-drop จาก jobs ที่มีอยู่ ให้ผู้ใช้ออกแบบ automation เองได้
   โดยไม่ต้องพิมพ์ทุกครั้ง — ต่อยอดจากระบบ jobs ปัจจุบัน

2. **Agent KPI & Performance Review** 🟢 — Director สรุปผลงานรายสัปดาห์ของแต่ละ agent
   (งานสำเร็จ/ล้ม, เวลาเฉลี่ย, ทักษะที่เรียนรู้) เป็นการ์ดน่ารักๆ — ต่อจาก dashboard

3. **Agent Mood/Energy system** 🟢 — agents มีพลังงานที่ลดเมื่อทำงานหนัก ฟื้นเมื่อพัก;
   เหนื่อยมากจะทำงานช้า/ขอพัก — เพิ่มชีวิตชีวาแบบ The Sims

4. **Smart routing** 🟡 — แทนที่จะระบุ agent เอง พิมพ์งานแล้วระบบเลือก agent ที่
   เหมาะที่สุดให้อัตโนมัติ (จาก skills/persona/ความว่าง)

5. **Long-running autonomous mode** 🔴 — สั่งเป้าหมายใหญ่ ("ทำเว็บ portfolio ให้เสร็จ")
   แล้ว Director วางแผนเป็น milestone, มอบงาน, ตรวจงาน, วนจนเสร็จ โดยรายงานความคืบหน้า
   เป็นระยะ — โหมด "ตั้งทิ้งไว้แล้วมาดูตอนเสร็จ"

## 🎨 โลกและภาพ

6. **Office Themes / Skins** 🟡 — ชุดธีมสำเร็จรูป (ไซไฟ, ญี่ปุ่น, โคซี่, ไซเบอร์พังก์)
   เปลี่ยนทั้งบรรยากาศ+สี+พร็อพในคลิกเดียว — ต่อยอดจาก editor (ข้อ 2 ที่กำลังทำ)

7. **Weather & seasons** 🟢 — ฝนตก/หิมะ/ใบไม้ร่วงนอกหน้าต่าง ตามฤดูจริงหรือสุ่ม
   ซิงค์กับ weather API ของเมืองผู้ใช้ก็ได้

8. **Photo mode / cinematic capture** 🟡 — กดปุ่มเดียวจัดมุมกล้องสวยๆ + ฟิลเตอร์
   ถ่ายภาพ/อัดวิดีโอสั้นออฟฟิศไปแชร์ (ของเล่นการตลาดชั้นดี)

9. **Pet & plant care** 🟢 — เลี้ยงแมว/หมา/ต้นไม้ ให้อาหาร/รดน้ำ มี mini-progression
   เพิ่มความผูกพัน

10. **Customizable avatar layers** 🟡 — แต่งตัว agents (หมวก/แว่น/ชุด/สีผม) จากระบบ
    layer ที่มีอยู่แล้ว — personalization ที่คนชอบมาก

## 🔌 การเชื่อมต่อและพลัง

11. **Plugin marketplace** 🟡 — หน้าเรียกดู/ติดตั้ง plugin จาก registry กลาง
    (ต่อจากระบบ plugins ข้อ 1 ที่กำลังทำ) — ระบบนิเวศจริง

12. **Calendar / Email / Notion sync** 🔴 — agents อ่านปฏิทิน Google, อีเมล, Notion
    ของจริงผ่าน MCP/OAuth — ผู้ช่วยจริงจัง ไม่ใช่แค่ของเล่น

13. **Screen awareness** 🔴 — agent "ดู" หน้าจอคุณ (screenshot ตามคำขอ) แล้วช่วยงาน
    ที่เห็นตรงหน้า — ผู้ช่วยที่อยู่ข้างๆ จริงๆ

14. **Webhook / IFTTT triggers** 🟡 — เหตุการณ์ภายนอก (PR ใหม่, ราคาหุ้น, อีเมล) →
    ปลุก agent ทำงาน; และ office → ยิง webhook ออก

15. **Mobile companion app** 🔴 — แอปมือถือดูสถานะออฟฟิศ + สั่งงาน (ต่อยอดจาก channels)

## ✨ ประสบการณ์ผู้ใช้

16. **Onboarding wizard** 🟢 — ครั้งแรกที่เปิด พาตั้ง key, จ้าง agent ตัวแรก, สอนพูด
    คุย — ลดกำแพงคนใหม่ (สำคัญมากตอนปล่อยจริง)

17. **Global command palette (Ctrl+K)** 🟢 — กดเรียกทุกอย่าง: คุย agent, เปิดโปรเจค,
    สั่งงาน, สลับธีม — ความเร็วระดับ power-user

18. **Achievements & milestones** 🟢 — ปลดล็อกความสำเร็จ ("100 งานแรก", "จ้างครบ 5 คน")
    + ออฟฟิศเติบโต/ตกแต่งเพิ่มตามเลเวล — gamification ที่ทำให้คนกลับมา

19. **Daily standup / briefing** 🟢 — ทุกเช้า Director รวบรวมสรุปเมื่อวาน + แผนวันนี้
    เป็นการ์ด/เสียงพูด — พิธีกรรมที่ทำให้รู้สึกเป็นออฟฟิศจริง

20. **Replay & time-lapse** 🟡 — ดูย้อนว่าวันนี้ออฟฟิศทำอะไรไปบ้างแบบเร่งเวลา
    (rebuild จาก journal ที่มีอยู่) — ทั้งสนุกและมีประโยชน์ debug

---

## โบนัส (คิดเผื่อ)

- **Team chat rooms** — ห้องแชทแยกหัวข้อให้ agents + คุณคุยรวมกัน
- **Cost budget & alerts** — ตั้งงบ/วัน เตือนเมื่อใกล้เต็ม (ต่อจาก stats)
- **Voice wake word** — "เฮ้ออฟฟิศ" เรียกโดยไม่ต้องกดปุ่ม
- **Shared cloud sync** — sync registry/projects ข้ามเครื่อง
- **Agent-built mini-games** — agents สร้างเกมเล็กๆ ใน rec room ให้เล่นจริง
