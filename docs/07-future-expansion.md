## 7.2 Training Academy (activated)

From flavor room → real capability:
- **Workflow lessons**: record a successful mission as a reusable workflow; the agent "attends class" while the system distills the trajectory into a template (prompt + tool plan). Graduation = certificate on the wall; the workflow appears in Quick Actions.
- **Prompt gym**: A/B improved system prompts against saved task suites in the simulation pods; metrics shown as exam scores.
- **Skill badges**: imported Claude Code skills / MCP tools show as badges agents physically receive at a small ceremony.

## 7.3 Replay Theater
A screening room off the Archive Library. Pick any past mission → the office **re-enacts it from the event journal** (time-scrubbing, 2×–32× speed): watch who did what, where it failed, what the meeting decided. Debugging as cinema; doubles as audit trail (exportable).

## 7.4 Agent Marketplace
- Hire pre-configured specialist agents (persona + prompt + toolset + sprite): "SEO Auditor," "Daily Standup Reporter," "Inbox Triager."
- Community-published, rated, sandbox-permissioned by default (worst keycard tier until promoted).
- Creator revenue share (see [Monetization](09-monetization.md)).

## 7.5 Multiplayer & Social
- **Office visits**: read-only guest links — show your office (and your live agent activity, scrubbed of sensitive text) to a friend/colleague.
- **Sister offices**: two users' offices connect via a "video-call wall" in the Meeting Room; agents can pass artifacts across (real file/PR handoffs between two people's agent fleets).
- **Company mode (teams)**: one shared office, each teammate's agents wear team colors; the Mission Control board becomes the team's real task board.

## 7.6 Mobile Companion
The daemon already runs headless — a phone app renders a "pocket lobby": status totem, notifications, approve/deny from anywhere, voice-dispatch a task to the office at home. Push: "Dev finished the deploy ✅."

## 7.7 Visual Workflow Designer
A drafting table in Mission Control: drag agent portraits onto a flow canvas (steps, branches, schedules — cron-friendly). Saved flows appear as framed blueprints; running a flow animates tokens moving along it in-world.

## 7.8 World Expansions (content packs)
- **New wings**: Finance Office (spend tracking per provider/model as an accounting room), Mail Room (email/inbox agents), Broadcast Studio (social/content agents publish from here), Rooftop Garden (pure cosmetic chill space).
- **Themes**: Night-city cyberpunk office, cozy woodland studio (Spiritfarer mood), retro 90s office, spaceship bridge. Themes reskin zones/lighting without changing semantics.
- **Seasonal live events**: first-party decorations + limited missions (Halloween office, snow at year-end) — retention beats.

## 7.9 Deep Integrations
- **Calendar/standup**: agents hold a real 9:00 standup in the Meeting Room summarizing yesterday/today across your tools.
- **GitHub/CI**: Dev Studio wall boards bind to real repos; a red pipeline summons a debugging agent automatically (policy-gated).
- **Home automation**: office lights/state mirrored to smart bulbs (the room glows amber when an approval waits — ambient computing beyond the screen).

## 7.10 Far-Horizon Bets
- **Embodied memory**: the Archive Library becomes a navigable knowledge map — walk shelves organized by embedding clusters; pull a book = open the source document.
- **Agent self-expansion**: agents propose hiring ("Ops is overloaded — hire a second researcher?") and office renovations based on real workload analytics.
- **OS-level presence**: agents "leave the office" through the front door when they act on your machine (browser pilot, file ops) and return with results — the office as the literal home of your computer's autonomy.
