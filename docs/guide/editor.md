# Office Editor — จัดออฟฟิศของคุณเอง 🎨

กดปุ่ม 🎨 บน header เพื่อเปิด Editor — วางเฟอร์นิเจอร์ ของตกแต่ง กำแพง และ
**นำเข้าโมเดล/รูปภาพ**ของตัวเองมาวางในออฟฟิศได้อิสระ โดยระบบหลัก (บรรยากาศ
กลางวัน/กลางคืน, แสง, เอฟเฟกต์, ตัวละคร) ยังอยู่ครบทุกอย่าง

## วิธีใช้

1. **เลือกของจาก palette** ด้านบน (โต๊ะ/เก้าอี้/ชั้นวาง/ต้นไม้/โคมไฟ/พรม/ฉากกั้น/กำแพง…)
2. **คลิกบนผัง** (มุมมองบนลงล่าง) เพื่อวาง — ผังนี้แทนพื้นที่จริงของออฟฟิศ
3. **ลากเพื่อย้าย** · **ดับเบิลคลิกเพื่อลบ**
4. คลิกเลือกของชิ้นไหน จะมีแถบปรับ **หมุน / ขนาด / สี** ด้านล่าง
5. กด **💾 บันทึกผัง** — โลกบนวอลเปเปอร์อัปเดต**ทันที**

## นำเข้าของตัวเอง

- **🖼 รูป (import)** — เลือกไฟล์รูป แล้ววาง → กลายเป็นป้าย/โปสเตอร์แขวนในออฟฟิศ
- **📦 โมเดล (import)** — เลือกไฟล์ `.glb` / `.gltf` / `.fbx` แล้ววาง → โมเดล 3D
  ของคุณโผล่ในโลกจริง

ไฟล์ที่นำเข้าถูกเก็บใน `workspace/uploads/`

## เบื้องหลัง (สำหรับนักพัฒนา)

- ผังเก็บเป็น `daemon/layout.json` — รายการ items แต่ละชิ้นมี
  `{ type, x, z, rot, scale, color, asset, w, d, h }` (พิกัดเป็นเมตรในโลกจริง)
- API: `GET /layout` · `POST /layout {items:[…]}` → daemon ยิง event
  `layout.changed`
- ฝั่ง Godot: `layout_loader.gd` ดึง `/layout` ตอนบูทและทุกครั้งที่บันทึก แล้ว
  สปอว์นของทับบนโลก procedural (เลเยอร์ `CustomLayout`) — ไม่แตะระบบหลัก

## เคล็ดลับ

- เริ่มจากของน้อยๆ ก่อน วางทีละชิ้นแล้วบันทึกดูผลบนวอลเปเปอร์จริง
- โคมไฟ (💡) เปล่งแสงจริง — ใช้แต่งบรรยากาศมุมห้องได้
- กำแพง/ฉากกั้น ปรับขนาด (scale) เพื่อทำเป็นห้องย่อยของตัวเองได้
