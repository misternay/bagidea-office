How it works

Talk like a boss

  • Type or speak into the CEO seat — that's you
  • The Director walks over, plans, and dispatches the team
  • Watch the hand-offs happen on the wallpaper in real time
  • Results report back; the final summary comes to you in person

Three layers, one exe

  • A zero-dependency Node daemon — the event hub + Claude adapter
  • A Godot 4 world embedded behind your desktop icons
  • A Rust shell with the chat head, overlay and tray
  • Agents are real headless claude sessions

Install in one line

Windows 11 · the installer sets up every dependency for you.

irm https://raw.githubusercontent.com/bagidea/bagidea-ai-agents-office/main/installer/install.ps1 | iex
1. Run the one-liner in PowerShell — it installs Git, Node, Rust, Godot & the Claude CLI
2. Run claude once to log in
3. bagidea start — your wallpaper goes to work 🏢
