Contribute

Publish your own plugin

Built something useful? A plugin can add a tool panel, agent commands, or whole new abilities — share it with every office.

1 · Build it

Start from the official template — it has a panel, a route and an agent command already wired. Shape it into your own plugin.

2 · Publish to GitHub

Push your plugin to a public GitHub repo with a plugin.json manifest. That repo is what people install from.

3 · Open a PR

Add an entry to the catalog via a pull request. We review every submission, then it appears here.
